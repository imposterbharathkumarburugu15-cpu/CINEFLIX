##  Features

###  Ticket Booking
- Users can browse movies/shows and book tickets instantly.
- Simple, clean, and efficient booking flow.

###  Watch-Later Mode
- Users can mark movies and shows to watch later.
- Saves selections for future access.

###  Download Movie Details
- Users can download all movie/show details easily.
- Useful for offline reference and sharing.

###  Direct Booking
- One-click booking design.
- No unnecessary steps or confusion.
